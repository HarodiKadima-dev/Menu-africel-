# 📲 Menu Africel — Simulador de Recargas e Dados

## 🎯 Objetivo do projeto
Criado **apenas para estudo e compreensão da lógica** por detrás dos sistemas de menu, recargas e pacotes de dados da operadora Africel.

✅ **Desenvolvido PURAMENTE em JavaScript** — sem uso de HTML nem CSS.

## ⚙️ Como funciona
- Estrutura completa de menus e sub‑menos hierárquicos
- Controlo do fluxo de navegação passo‑a‑passo
- Leitura e validação das opções escolhidas
- Respostas e sequências simuladas diretamente pelo código
- Funciona por lógica de programa, sem interface visual de página

## 🧠 O que aprendi aqui
- Organizar fluxos de decisão
- Usar funções, ciclos e condições de forma inteligente
- Gerir escolhas e voltar aos níveis anteriores
- Estruturar sistemas parecidos com os que existem nos serviços reais

## 🛠️ Tecnologia utilizada
🔹 **JavaScript — exclusivamente**  
Todo o funcionamento, estrutura e comportamento do sistema estão escritos só nesta linguagem.
